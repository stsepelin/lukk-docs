# lukk-docs

Unified documentation for **lukk** (the Laravel package) and **lukk-js** (the TypeScript / Nuxt client), built with [VitePress](https://vitepress.dev).

Both halves are documented as one story, organized by topic — each feature page carries a server section and a client section.

```bash
npm install
npm run docs:dev      # local preview at http://localhost:5173/lukk/
npm run docs:build    # static build → docs/.vitepress/dist
```

Deployed to <https://stsepelin.github.io/lukk> (supersedes the former per-repo docs sites).
